# ESP32-CAM-FOTOPAST
Tato práce se zabývá návrhem a realizací budget fotopasti založené na mikrokontroléru ESP32-CAM. Zařízení využívá PIR senzor pro detekci pohybu, po které následuje pořízení snímku pomocí kamerového modulu OV2640 a jeho uložení na microSD kartu. Projekt demonstruje využití IoT technologií v praxi s důrazem na minimalizaci nákladů a energetickou efektivitu.
